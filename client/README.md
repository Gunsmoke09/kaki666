# Handcraft Tutorial Frontend (React App)

## Purpose

The **Handcraft Tutorial Frontend** is a React-based web application that provides users with a clean and intuitive interface to explore handcraft tutorials, categories, and materials.

It interacts with a backend API to handle user registration, authentication, and content management. The frontend is designed to be a responsive and user-friendly client for the Handcraft Tutorial API.

---

## Features

- **User Authentication:** Register and log in to access protected features.
- **Tutorial Management:** View, create, update, and delete tutorials.
- **Category Management:** Browse and manage craft categories.
- **Material Management:** Browse and manage tools and resources.
- **Tutorial Detail Pages:** View full tutorial instructions and linked data.
- **Search & Sorting:** Quickly find relevant content.
- **Pagination:** Organised browsing for larger datasets.
- **Responsive Design:** Works across desktop, tablet, and smaller screens.

---

## Dependencies

The **Handcraft Tutorial Frontend** relies on the following dependencies listed in `package.json`:

- **react** – JavaScript library for building user interfaces.
- **react-dom** – DOM rendering for React.
- **react-router-dom** – Client-side routing and navigation.
- **vite** – Frontend build tool and development server.
- **@mantine/core** – UI component library.
- **@mantine/hooks** – Mantine React hooks.
- **@mantine/dates** – Date-related UI components.
- **@mantine/notifications** – User notifications and alerts.
